## Introduction
Welcome to the **Angular 17 & .NET 8 ERP Application** project!  
This project is a comprehensive **Enterprise Resource Planning (ERP)** application developed using modern software development principles and based on real-life business scenarios.

The project was developed as part of the Udemy course **“.NET 8 and Angular 17 ERP Application”**. This course aims to help me gain experience in backend development, frontend development, and understanding the overall structure of an ERP system.

---

## Project Overview
This ERP application is designed as a scalable and sustainable system that manages corporate processes such as users, roles, products, stock, and similar business operations.

Throughout the project:
- A RESTful Web API was developed using .NET 8  
- Clean Architecture & CQRS architecture was implemented  
- A modern and modular frontend was built with Angular 17  
- Design patterns commonly used in real-world projects were applied  

---

## Features

### Backend
- Clean Architecture (Layered Architecture)
- CQRS and MediatR
- Entity Framework Core (Code First)
- MySQL
- FluentValidation
- Result Pattern
- AutoMapper

### Frontend
- Angular 17
- AdminLTE dashboard theme
- Backend API integration

---

## Technologies Used
This project leverages modern and reliable technologies to ensure high performance, security, and maintainability:

- **.NET 8** – .NET platform for building modern web applications  
- **Entity Framework Core (Code First)** – Efficient data management with MySQL using a flexible ORM  
- **MySQL** – A fast and reliable relational database solution  
- **FluentValidation** – A powerful library for validating user input with clear and maintainable rules  
- **Frontend (Angular 17)** – A modern framework for building dynamic and modular single-page applications  
- **UI Theme (AdminLTE)** – A ready-to-use admin dashboard template for creating clean and responsive user interfaces
